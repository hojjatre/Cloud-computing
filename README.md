# cloud-computing

## Requirements
1. Install django:
    `pip install django`
2. Instal django rest framework:
    `pip install djangorestframework`
3. Install numpy:
    `pip install numpy`
4. Install matplotlib:
    `pip install matplotlib`

## Convert CSV to Sqlite3
1. `sqlite3 db.sqlite3`
2. `.mode csv`
3. `.import vgsales.csv app_vgsales`
